## 주요 변경 사항

### 3. OSM → 카카오맵 JS API 전환
- Leaflet 완전 제거, 카카오맵 SDK로 교체
- `KAKAO_JS_API_KEY` 추가 (`backend/.env`)
- `/api/config` 엔드포인트로 JS 키 프론트에 전달
- 그림자 GeoJSON → `kakao.maps.Polygon` 배열 변환
- 경로 → `kakao.maps.Polyline`, 마커 → `kakao.maps.CustomOverlay`
- 카카오 개발자 콘솔에서 `http://localhost:3000` 허용 도메인 등록 필요

### 4. 장소 검색 드롭다운 추가
- `/api/search` 엔드포인트 추가 (결과 최대 5개)
- 현재 지도 중심 기준 거리순 정렬 (`x`, `y`, `sort=distance` 파라미터)
- 300ms 디바운스로 타이핑 중 자동 검색
- 결과 선택 시 해당 좌표 고정 (같은 이름 다른 지점 오인 방지)

### 5. 횡단보도 제약 강화 (`route_model.py`)
- **OSM 그래프에서 `highway=crossing` 노드 자동 추출** → 서울 API 없어도 동작
- 서울 API 횡단보도 + OSM 횡단보도 병합 사용
- shapely `crosses()` 로 보행 엣지가 실제로 차도를 가로지르는지 기하학적 검사
  - 기존: 도로 엣지 중점 근처에 횡단보도 있는지만 체크 → 도로 옆 인도도 패널티 맞는 버그
  - 수정: 인도는 패널티 없음, 실제 횡단 엣지만 감지
- `footway=crossing` 명시 엣지는 패널티 면제
- 적용 도로: `primary`, `secondary`, `tertiary` 및 link류만 (골목·이면도로 제외)
- 패널티: 60분 (사실상 비횡단보도 도로 횡단 금지)

### 6. 그림자 가중치 조정
- 그늘 가중치 감소율: 0.8 → 0.3 (기존엔 그늘 찾으러 5배 이상 돌아가는 문제)
- 이제 동일 거리 대비 최대 43% 우선, 2배 이상 돌아가면 선택 안 함

### 7. 시간 입력 UX 개선
- `#time-display` span → input으로 변경, 직접 타이핑 가능 (`HH:MM` 형식)
- 그림자 폴리곤 중첩 문제 수정: AbortController로 이전 fetch 취소 후 새로 로드
